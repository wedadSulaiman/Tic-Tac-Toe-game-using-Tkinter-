# Tic-Tac-Toe Game (Python Tkinter GUI)

This project is a Tic-Tac-Toe game developed using Python and the Tkinter library.  
It provides a graphical user interface that allows a player to play either against the computer or against another player.

---

## Features

- Graphical User Interface using Tkinter
- Player vs Computer mode
- Player vs Player mode
- Option to choose symbol (X or O)
- Computer makes logical moves
- Automatic detection of win and draw conditions
- Option to play again after the game ends
- Simple and user-friendly interface

---

## Technologies Used

- Python 3
- Tkinter (standard Python GUI library)
